# 📋 Wolf Tasks - מערכת לניהול משימות בצוות

מערכת לניהול משימות בצוותים, בדומה ל-ClickUp, הבנויה ב-Angular 20. המערכת מאפשרת עבודה בצוותים, ניהול פרויקטים, וניהול משימות עם לוח קנבן.

## 🎯 תכונות עיקריות

* **ניהול משתמשים:** התחברות והרשמה מאובטחת.
* **צוותים:** יצירת צוותים והוספת חברים.
* **פרויקטים:** ניהול פרויקטים לכל צוות.
* **לוח משימות (Kanban):** משימות בסטטוסים: לביצוע, בתהליך, הושלם.
* **עיצוב:** ממשק משתמש מודרני ונוח.

## 🛠️ טכנולוגיות

* **Frontend:** Angular 20, TypeScript, Signals, RxJS
* **Backend:** Node.js, Express, SQLite (Server provided separately)
* **Security:** JWT Authentication

## 🚀 הרצה מקומית

1.  **התקנה:**
    ```bash
    npm install
    ```
2.  **הרצת השרת (חובה להריץ במקביל):**
    ודא ששרת ה-Node.js רץ בפורט 3000.
3.  **הפעלת האנגולר:**
    ```bash
    ng serve
    ```
    הכנס לדפדפן בכתובת: `http://localhost:4200`

## 🌐 פרטי גישה (API)

המערכת מתקשרת עם השרת בכתובת `http://localhost:3000/api`.
הנתיבים המרכזיים:
* `POST /auth/login`
* `GET /teams`
* `GET /projects`
* `GET /tasks`
